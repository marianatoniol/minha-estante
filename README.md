# Minha Estante 📚

Sua biblioteca pessoal inteligente com classificação por tropes usando IA.

## Como publicar no Vercel

1. Suba este projeto para o GitHub
2. No Vercel, importe o repositório
3. Clique em "Deploy"
4. Pronto! Acesse pelo link gerado

## Como instalar no celular (PWA)

- **iPhone**: Abra o link no Safari → toque no botão de compartilhar → "Adicionar à Tela de Início"
- **Android**: Abra o link no Chrome → toque nos 3 pontinhos → "Instalar aplicativo"

## Configuração

Após instalar, vá em Config e cole sua chave da API Anthropic para ativar a classificação automática por tropes.
